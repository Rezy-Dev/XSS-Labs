# XSS-Labs

## Setup
```bash
git clone https://github.com/Rezy-Dev/XSS-Labs.git && cd XSS-Labs
```

## Start PHP Server
```bash
❯ php -S localhost:1337                 
[Sat Jun  7 23:51:35 2025] PHP 8.3.6 Development Server (http://localhost:1337) started
```
Visit http://localhost:1337/ and start pwning!
